# Changelog

## 0.1.0

- Initial release
- 21 type guard functions with TypeScript type narrowing
- Primitives: `isString`, `isNumber` (excludes NaN), `isBoolean`, `isBigInt`, `isSymbol`, `isFunction`
- Nullish: `isUndefined`, `isNull`, `isNullish`
- Objects: `isPlainObject`, `isArray`, `isTypedArray`, `isDate`, `isRegExp`, `isError`, `isMap`, `isSet`
- Async: `isPromise`, `isIterable`, `isAsyncIterable`
- Utility: `isEmpty`
